# dcbot

A minimal Discord bot built with Python.

## Features

- Connects to Discord using a bot token
- Listens to messages in a server
- Responds to simple commands (e.g. `ping -> pong`)

## Tech Stack

- Python 3.10+
- discord.py
- python-dotenv
